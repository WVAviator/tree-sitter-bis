# BIS Command Reference

## About This Reference

This reference contains information on commands available on all Business Information Server (BIS) systems.

> For commands that perform system administration tasks, see the **Administration Guide**.

---

## Organization

Commands are arranged alphabetically by **call** — the string of characters used to invoke the command. A command may be:

- A **manual function or run**
- A **statement** used by application designers to write runs that automatically perform sequences of activities

### Statements vs. Functions

If a call in a topic title is preceded by an at sign (`@`), it is a **statement**. If you are not an application designer, you can ignore these topics.

Some topics cover two or more calls together (e.g., `F` and `@FND` — Find), meaning the topic contains information relevant to both general users and application designers. Within such topics, areas flagged with `@` are intended for application designers.

# NAMELIST

Defines the current session's Active Namelist. Once set, you can use `NAMELIST` as a data name with Search, Locate, or other functions to process your Namelist data.

---

## Syntax

```
NAMELIST,c,d,r
```

`c,d,r` is treated as the Namelist Definition Report. After entering the command, a screen is displayed where you can enter the information for the body of the Namelist.

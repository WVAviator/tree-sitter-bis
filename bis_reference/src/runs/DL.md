# DL — Drawer List

## Overview

Displays a list of drawers you are registered to access. From the list, you can select a drawer to display its reports. If a cabinet number is specified, lists accessible drawers in that cabinet instead.

> **Note:** Specifying another cabinet does not switch your current cabinet.

Similar to the Report key, but allows specifying a different cabinet on the control line.

---

## Syntax

```
DL [c]
```

### Parameters

| Field | Description |
|-------|-------------|
| `c` | Cabinet number to list drawers from, if different from your current cabinet. |

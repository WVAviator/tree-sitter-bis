# VIEW — Create Temporary Format

## Overview

Changes the selection of report fields displayed, allowing non-adjacent fields to be placed side by side. For example, a field in columns 20–30 can be displayed next to a field in columns 85–90. A temporary format is similar to a permanent format but applies to the current display only.

> **Note:** To create a permanent format, contact your administrator.

The report for which to create the temporary format must be on display. If `VIEW` is unable to display all report fields due to report width and screen size, the undisplayed fields cannot be included in the temporary format.

> **Note:** A workstation must be used to view all fields of a wide report.

---

## Syntax

```
VIEW
```

---

## Procedure

1. Type `view` on the control line and transmit. A mask containing the report headings is displayed.
2. Type an `x` in each field to include in the new format and transmit.

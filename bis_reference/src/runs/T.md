# T — Cabinet Table of Contents

## Overview

Displays a list of available drawers in your cabinet. Drawer A is accessible from all cabinets and is not included in the list.

*(Windows / Linux / UNIX)* The list includes only those drawers accessible to your security group, as registered by the administrator.

Each line in the table of contents describes one drawer and contains:

- Drawer name
- Drawer letter
- Current cabinet number
- `W` if you have write privileges for the drawer, or `R` if you have read-only privileges

---

## Syntax

```
T
```

---

## Guidelines

With the table of contents on display, you can:

- Tab to a selection and transmit to select a drawer. The command displays a list of reports in the drawer, from which you can select a report.
- Press **Tasks** to perform processing on the drawer.

# UNDO

## Overview

Restores your data to its state before the execution of the last command. Common uses include:

- Restore a report you just deleted.
- Undo the last update made to a report.
- Restore a previously displayed result.

> **Note:** A command for which the Undo operation is available must have been executed first.

---

## Syntax

```
UNDO
```

Or press **Undo**.

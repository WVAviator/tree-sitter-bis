# TW — Terminal Width

> **OS 2200 only**

## Overview

Sets the display width of SVT-1120, SVT-1121, or PWT-120 terminals to 80 or 132 columns. The command toggles the display width between the two values.

> **Note:** The terminal width must also be set in the control page of the terminal.

---

## Syntax

```
TW {80|132}
```

where `{80|132}` is the number of columns for the terminal width.

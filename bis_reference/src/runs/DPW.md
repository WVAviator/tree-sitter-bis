# DPW — Drawer Password

> **2200 only.**

## Overview

Unlocks a drawer protected by a password assigned by the administrator. Access is maintained for up to two drawers simultaneously, until the active screen is displayed.

> **Note:** This function restricts manual function users only — statements can overwrite reports in password-protected drawers without using `DPW`.

---

## Syntax

```
DPW
```

### Parameters

None. Must be used via the control line — a menu selection is not available.

---

## Usage

Enter `DPW` on the control line. The system displays a `DPW Enter Drawer Password` prompt. Type the drawer password in the `DPW` field and transmit. The function repeats the password prompt automatically until a valid password is entered.

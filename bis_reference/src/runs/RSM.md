# RSM — Resume

## Overview

Use the `RSM` command to continue execution of a command after it has been suspended by a display. This is particularly useful with the Find and Locate commands, as it allows you to continue a search after the system displays each find.

---

## Syntax

```
RSM
```

Or press **Resume** if it is displayed on the function key bar. The Resume key appears when you are using a command for which processing can be resumed.

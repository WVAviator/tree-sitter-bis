# RL (Report List)

## Overview

Displays a list of reports you can access. Similar to the Report key, but allows you to specify a different drawer and cabinet.

If no drawer is specified, the command displays a list of drawers in your current cabinet from which you can select to view reports. If a drawer (and optionally a cabinet) is specified, it displays a list of accessible reports in that drawer.

> **Note:** Specifying another cabinet does not switch you to that cabinet.

---

## Syntax

```
RL [dc]
```

`dc` is the drawer letter and cabinet number, if different from the current cabinet, from which to display a list of reports.

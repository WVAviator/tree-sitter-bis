# RETN (Return Result from Remote)

## Overview

Returns a result from a remote site to the local site.

---

## Syntax

```
RETN
```

---

## Example

Connect to a remote MAPPER using the remote function key. Generate a result report, and with it displayed enter `RETN` on the command line. This returns both control and the result to your local system, where it appears in your current cabinet as a result report.

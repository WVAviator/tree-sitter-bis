# PCX

## Overview

The PCX utility transfers data between a report and a DOS file. Data can be transferred in either direction — MS-DOS to Business Information Server, or Business Information Server to MS-DOS.

This utility requires a workstation session using one of the following clients:

- Graphical Interface for Business Information Server
- Business Information Server for Microsoft Windows Client

> **Note:** If the session is using either of these clients, the reserved word `WS$` (workstation flag) will equal `1`.

---

## Usage

For additional information, launch the utility and access its built-in help:

1. Enter `PCX` and press Enter.
2. From the menu bar or the help button, select **Help**.

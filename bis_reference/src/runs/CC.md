
# CC (Column Count)

Use the **Column Count (`CC`)** command to display column numbers at the top of the screen or report (lines 2 and 3). Displays correct column numbers regardless of the current format.

## Column Count Control Line Format

```
CC
```

## Column Count Guidelines

- With column counts on display, you can perform SOE updates or scroll through your report.
- If you shift or change formats, the column counts are removed from the display.

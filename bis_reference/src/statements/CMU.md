# @CMU — Commit Updates

## Overview

Saves all updates made to reports for which updates were deferred by a preceding [`@DFU`](DFU.md) (Defer Updates) statement. Also releases update control for those reports.

> The reports to be committed must have been specified in a preceding `@DFU` statement.

---

## Syntax

```
@CMU .
```

### Parameters

None.

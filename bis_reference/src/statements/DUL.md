# @DUL — Defer Unlock

## Overview

Removes a non-updated report from the deferred-update list.

---

## Syntax

```
@DUL,c,d,r,lab .
```

### Parameters

| Field | Required | Description |
|-------|----------|-------------|
| `c,d,r` | Required | Report to remove from the deferred-update list. |
| `lab` | Required | Label to branch to if the report does not exist. |

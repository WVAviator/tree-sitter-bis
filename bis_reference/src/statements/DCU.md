# @DCU — Decommit Updates

## Overview

Discards all updates made to reports for which updates were deferred by a preceding [`@DFU`](DFU.md) (Defer Updates) statement. Also releases update control for those reports. Updates to other reports are not affected.

> The reports to be decommitted must have been specified in a preceding `@DFU` statement.

---

## Syntax

```
@DCU .
```

### Parameters

None.

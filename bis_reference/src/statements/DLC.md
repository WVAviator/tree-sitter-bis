# DLC — Display Line Count

## Overview

Displays line numbers along the right side of the screen. Press **Resume** to remove them.

> A report must be on display.

---

## Syntax

```
DLC[,S]
```

### Parameters

| Field | Description |
|-------|-------------|
| `S` | Displays both the report line count and the screen line count. Default (no option) = report line numbers only. |

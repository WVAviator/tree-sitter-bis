# X and @XIT — Sign Off User Session

## Overview

Signs off and returns to the sign-on screen. Using the `@XIT` statement also terminates the run in which it is executing before returning to the sign-on screen.

---

## Syntax

### Control Line
```
X
```

### Statement
```
@XIT .
```

---

## Procedures

### Using the X Command

1. Move the cursor to the control line (or home position if the active screen is displayed).
2. Type `x` and transmit to end your user session.

### Using the SgnOff Key

1. Press the **Return** function key until you reach a screen with **SgnOff** on the function key bar.
2. Press **SgnOff** to end your user session.

# LANG and @LNG — Language

## Overview

Changes the text of system messages to another language (for example, French, if available on your system). The assigned language remains in effect for future system messages until the user signs off or selects another language.

---

## Syntax

```
LANG [n]
@LNG,n .
```

### Parameters

| Field | Platform | Description |
|-------|----------|-------------|
| `n` | Windows / Linux / UNIX | Language number. Valid entries: `1`–`8`. `1` = primary language at your site. Refer to the input screen for the list of languages available at your site. |
| `n` | 2200 | Language number. Valid entries: `0`–`7`. `0` = primary language at your site. Refer to the input screen for the list of languages available at your site. |

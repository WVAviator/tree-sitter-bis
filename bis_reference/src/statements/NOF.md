# @NOF — Network Off

## Overview

Signs off the remote site after executing other network statements such as [`@NRD`](NRD.md) (Network Read) and [`@NWR`](NWR.md) (Network Write). Closes the communications connection with the remote host.

Use [`@NET`](NET.md) (Network Sign-On) to connect to the remote site before using this statement.

---

## Syntax

```
@NOF .
```

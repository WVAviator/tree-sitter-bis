A statement used by application designers to write runs
that automatically perform sequences of activities.
In a topic title, if a call is preceded by an at sign (@), it is a
statement. If you are not an application designer, you can
ignore these topics.

Some titles contain two or more calls, such as F and @FND
(Find). A topic such as this contains information of interest
to users whether they are application designers or not.
Additionally, within a topic such as this, you will see some
areas ﬂagged with at signs (@). Again, if you are not an
application designer, you can ignore these areas.

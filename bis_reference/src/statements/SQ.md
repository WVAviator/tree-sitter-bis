# @SQ

> *2200 only*

## Overview

Use the `@SQ` statement to restart printing from a run.

---

## Syntax

```
@SQ[,lab] sn .
```

### Parameters

| Field | Required | Description |
|-------|----------|-------------|
| `lab` | Optional | Label to go to if the station table does not exist. |
| `sn` | Required | Station number of the printer. |
